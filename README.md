# Breast Cancer Risk Prediction & Interactive Dashboard

This project provides a comprehensive solution for **Breast Cancer Risk Prediction** using advanced Machine Learning techniques and **Interactive Data Visualization**. It combines predictive modeling with a professional dashboard to offer insights into key medical parameters and diagnosis patterns.

## 🚀 Overview

The primary objective of this repository is to analyze breast cancer diagnostic data and build robust classification models (Logistic Regression and Random Forest) to predict the likelihood of a diagnosis being Benign or Malignant. The results are further explored through an interactive Power BI dashboard, enabling data-driven decision-making and intuitive exploration of clinical metrics.

## 🛠️ Key Features

- **Predictive Modeling**: Implementation and comparative analysis of Logistic Regression and Random Forest classifiers.
- **Data Preprocessing**: Comprehensive cleaning, encoding, and feature scaling for optimal model performance.
- **Feature Exploration**: Analysis of critical clinical features such as:
  - Radius, Texture, Perimeter, and Area.
  - Compactness, Concavity, Concave Points, and Symmetry.
  - Demographic factors: Age, Gender, Family History, and Dietary Habits.
- **Interactive Visualization**: A professional Power BI dashboard (`.pbix`) that tracks key diagnostic indicators and risk factors.

## 💻 Technologies Used

- **Data Science**: Python (Pandas, NumPy, Scikit-Learn, Matplotlib, Seaborn)
- **Environment**: Jupyter Notebook
- **Visualization**: Power BI Desktop
- **Version Control**: Git

## 📂 Repository Structure

- `Breast_Cancer_Risk_Analysis.ipynb`: Detailed exploratory data analysis and initial modeling.
- `Comparative_Analysis_of_Logistic_Regression_and_Random_Forest...ipynb`: Performance evaluation and comparison of machine learning models.
- `Breast Cancer Dashboard.pbix`: The interactive Power BI dashboard file.
- `Breast Cancer Dashboard.png`: A high-resolution screenshot of the dashboard.
- `breast_cancer_data.csv`: The clinical dataset used for training and visualization.

---

### 📝 Note on Dataset
*The dataset used in this project consists of synthetic clinical records generated for research, training, and demonstration purposes. While it reflects realistic clinical parameters, it does not represent actual patient data and is intended solely for academic and technical application.*
